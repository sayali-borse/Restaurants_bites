🍽️ Restaurant Web App

A modern and responsive Restaurant Web Application where users can explore the menu, view food items, and experience a clean restaurant UI. This project demonstrates front-end web development skills using HTML, CSS, and JavaScript.

🚀 Features

Responsive restaurant website design

Attractive food menu layout

Navigation bar with smooth scrolling

Menu section with different food categories

Contact section for customer inquiries

Mobile friendly design

Clean UI/UX

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and responsive design

JavaScript – Interactive functionality

Git & GitHub – Version control and project hosting
